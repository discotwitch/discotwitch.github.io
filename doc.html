Most commands are optional. The only commands that you need for setup are !dt follow and !dt channel.

Twitch Commands

!dt follow user,user,user...

Choose Twitch users to get alerts for.
You can follow multiple users at once, separate them with commas (without spaces).

Examples:

!dt follow xqcow
> follow xqc (start getting alerts for them)

!dt follow xqcow,esl_csgo
> follow xqc and esl_csgo

!dt unfollow user,user,user...

Unfollow users to stop getting alerts for them.
Use "unfollow all" to stop following everyone at once.

Examples:

!dt stop getting alerts for xqcow,esl_csgo
> unfollow xqc and esl_sc2

!dt status user

Show the online status of a provided user.
You can also use a plain status command to list all your followed users' live statuses.

Examples:

!dt status
> list which followed users are live

!dt status esl_sc2
> see if twitch.tv/esl_sc2 is live

!dt clips user

View the all-time top clip of a twitch user.

Examples:

!dt clips xqcow
> see xqc's top clip

Discord Commands

!dt message "on-live message"

Change the on-live alert message.

You can use the following modifiers:
<user> inserts Twitch username
<link> inserts Twitch link
<title> inserts stream title
<game> inserts the game they're playing
<br> inserts newline

Note:

Using this command is optional, as the bot has a default message

Examples:

!dt message "<user> is live! <link> <br> Playing <game>"

> sets alert message to:
Joe is live! https://twitch.tv/joe
Playing Minecraft

!dt channel server-channel

Choose where the bot will post alerts.

Examples:

!dt channel #on-live
> tell bot to post alerts in #on-live channel.

!dt role server-role

Set the required role to send commands to DiscoTwitch.

Default role is @everyone, meaning anyone can follow/unfollow users, change the post channel,
alert message, and nickname.

Examples:

!dt role @Admin
> set role to @Admin

!dt nickname name

Change the nickname of the bot in your server.

Examples:

!dt nickname Live-Bot
> set DiscoTwitch's nickname to "Live-Bot"

!dt settings options

Customize misc settings.

Options:

0 - mute (enabling other options will override this)
1 - get on-live notifications (enabled by default)
2 - get title change alerts
3 - get offline alerts
4 - edit on-live alerts when channels go offline (enabled by default)
5 - get VOD links in offline alerts
6 - edit VOD links into on-live messages when channels go offline

Examples:

!dt settings 13
!dt settings 31
> get only on-live & offline alerts (the order of the numbers doesn't matter)

Command Shorthand

Some commands can be shortened for easier usability.

!dt help > !dt h

!dt follow > !dt f

!dt unfollow > !dt u

!dt channel > !dt ch

!dt message > !dt msg

!dt status > !dt s

!dt setting > !dt set

!dt nickname > !dt nick
